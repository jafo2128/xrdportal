XRD Portal

This is a portal for an XRD department at a University. It just queues samples and manages files as well as notices.

View on GitHub Download .zip Download .tar.gz

The project is written in CodeIgniter with a MySQL database as well as Bootstrap for UI. It also uses the JMol library for rendering cif files and graphic models in a browser Java applet.
